## 标题
Markdown中标题如果想定义一个标题，可以在前面加上#关键字。
1-6个#号对应表示1-6级标题，标准Markdown语法#与正文之间有空格。
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 列表

## 文本样式
Markdown支持4中文本样式分别是：加粗、斜体、删除线和加粗且斜体。
```
*斜体*
**加粗**
**_加粗且斜体_**
~~删除线~~
```

*斜体*
**加粗**
**_加粗且斜体_**
~~删除线~~

## 引用
只需要在文本前加入 > 这种尖括号（大于号）即可，引用之间也可以嵌套。

> 这是一个引用
> > 这也是一个引用
> >
> > > 这还是一个引用

## 引用代码
引入一个代码用倒引号`括起来
引用一段代码用三个倒引号括起来

```
引用一段代码
int a = 10;
int b = 20;
int c = a + b;
cout<<c<<endl;
```

上面可以直接在三个倒引号后面加入引用代码的语言类型。编辑器会对应的进行渲染。


``` c++
引用一段代码
int a = 10;
int b = 20;
int c = a + b;

```

## 转义符
可以用反斜杠来插入一些在语法中有其它意义的符号，如：可以在星号的前面加上反斜杠。
```
\*literal asterisks\*
```

*literal asterisks*

反斜杠支持下面的字符
```
\   反斜杠
`   反引号
*   星号
_   底线
{}  大括号
[]  方括号
()  括号
#   井字号
+    加号
-    减号
.   英文句点
!   惊叹号
```
